About Fuel Blending Optimization

Overview

The Fuel Blending Optimization Web Application provides a user-friendly interface to the powerful Fuel Blending Optimization Package. This application is designed to help refineries and blending operations achieve optimal results through sophisticated mathematical optimization techniques.

Core Capabilities

  • Blend Optimization

    Find the least-cost blend recipe that meets all specified quality constraints for various fuel grades.

  • Candidate Evaluation

    Assess the economic impact and feasibility of user-defined blend recipes compared to the optimal solution.

  • Economic Analysis

    Calculate shadow prices and neutral prices to understand the economic impact of constraints and component costs.

  • Flexible Data Management

    Support for various file formats (Excel, CSV, JSON, YAML) for input data and results.

Technical Details

The application leverages these key technologies:

  • Backend: Flask (Python web framework)
  • Frontend: Bootstrap 5, JavaScript, Chart.js
  • Optimization Core: SciPy optimization algorithms (SLSQP, trust-constr, COBYLA)
  • Data Processing: Pandas, NumPy

Development Philosophy

This application was designed with the following principles in mind:

  • User-Centric Design: Intuitive interfaces for non-technical users while offering advanced functionality for experts.
  • Performance: Optimized for handling complex optimization problems efficiently.
  • Flexibility: Support for a wide range of input data formats and optimization scenarios.
  • Visualization: Clear graphical representation of results for better understanding and decision-making.

Technical Architecture

System Architecture

The web application follows a layered architecture pattern:

  1. Presentation Layer: HTML templates with Bootstrap styling and JavaScript for interactivity
  2. Application Layer: Flask routes and controllers that handle HTTP requests and responses
  3. Service Layer: Python modules that interact with the Fuel Blending Optimization package
  4. Core Optimization Layer: The underlying Fuel Blending Optimization package with its models and algorithms
