Getting Started

To use the Fuel Blending Optimization application, you'll need:

  • Component data file (Excel/CSV) with properties and costs
  • Grade specifications file (Excel/JSON) with target properties
  • Optional: Additives file (Excel/YAML) with cost information

Click on one of the cards above to get started with your optimization tasks.

Key Features

Utilizes sophisticated mathematical optimization techniques to find the minimum cost blend that satisfies all quality specifications.

Provides shadow prices and neutral prices to help understand the economic impact of constraints and component costs.

Supports various file formats (Excel, CSV, JSON, YAML) for input data and provides results in CSV or JSON format.

User-friendly web interface that simplifies the optimization process and visualizes results effectively.
